# Hello World

  Example app with node.js, express, mongoose, hogan, and coffeescript

  Uses capistrano for deploy to EC2 server.
  
## Setup

    npm install
    mongod --dbpath /Users/jeff/development/tools/mongo-data #replace with your path

## Starting app

    nodemon app.js
